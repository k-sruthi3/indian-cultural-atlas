About Indian Cultural Atlas

Indian Cultural Atlas is a web platform designed to explore and preserve the rich cultural heritage of India. It aims to digitally document the diversity of Indian states and districts, making cultural knowledge easily accessible to students, researchers, and enthusiasts.

The platform allows users to discover famous food, festivals, temples, traditional dances, folk arts, and heritage monuments across India. It also includes a submission system where users can contribute cultural information, which is verified by administrators.

Built using Django, SQLite, Bootstrap 5, HTML, CSS, and JavaScript, the system is designed to be scalable, user-friendly, and educational.

Features

  • Explore Indian States and Districts
  • Discover Famous Food, Festivals and Temples
  • Learn Traditional Dances and Folk Arts
  • View Cultural Monuments and Heritage Sites
  • Submit Cultural Information for Admin Review

Technology Used

  • Django
  • SQLite
  • Bootstrap 5
  • HTML, CSS, JavaScript

Future Enhancements

  • Multilingual support for regional languages
  • Interactive cultural map of India
  • AI-based cultural recommendations
  • Mobile-friendly Progressive Web App (PWA)
  • Integration with tourism and heritage APIs
